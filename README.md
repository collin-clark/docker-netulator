New and improved IP calculator that is now on Docker!

Enter a dotted decimal (198.51.0.157 255.255.225.224) or CIDR (198.51.0.157/27) an receive lots of information about that subnet. 

- Network ID
- Broadcast Address
- Size of the subnet
- Subnet Mask
- CIDR
- Wildcard Mask
- Host Size & Range
- Child Subnets
- CIDR Range
- and more!
